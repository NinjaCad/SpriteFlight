## 🚀 *Sprite Flight*

### 🎮 Overview

*Orbital Escape* is a fast-paced 2D survival game where you control a small spacecraft inside a chaotic arena filled with moving debris. Your goal is to survive as long as possible while avoiding collisions.

---

## 🕹️ How to Play

* **Hold left mouse button** to move toward your cursor
* Your ship automatically faces the direction it moves
* Avoid all objects
* Your **score increases over time**
* If you crash, click **Restart** to try again

---

## ⚙️ Features

* **Physics-based movement** using forces
* **Mouse-controlled navigation** (direction + thrust)
* **Score system** that increases over time
* **Collision effects** (explosions and bounce particles)
* **Randomized obstacles** (size, speed, and spin)
* **UI Toolkit** for score display and restart button
* **Scene reload system** for restarting the game
